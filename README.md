# 💬 ZenexChat

Продвинутая система чата

## 📦 Установка
1. Скачайте `ZenexChat.jar`
2. Поместите в папку `plugins`
3. Перезапустите сервер

## 🎯 Команды
| Команда | Описание | Право |
|---------|----------|-------|
| `/g <сообщение>` | Глобальный чат | `zenexchat.global` |
| `/l <сообщение>` | Локальный чат | `zenexchat.local` |
| `/voice <сообщение>` | Голосовое сообщение | `zenexchat.voice` |
| `/vc включить` | Включить голосовой чат | `zenexchat.vc` |

## 🔐 Права
| Право | Описание |
|-------|----------|
| `zenexchat.global` | Доступ к глобальному чату |
| `zenexchat.local` | Доступ к локальному чату |
| `zenexchat.voice` | Доступ к голосовым сообщениям |
| `zenexchat.vc` | Доступ к голосовому чату |
| `zenexchat.color` | Использовать цвета в чате |
| `zenexchat.gradient` | Использовать градиенты |
| `zenexchat.admin` | Административные права |

## 🎨 Особенности
- Градиенты и HEX цвета
- Голограммы над головой
- Интеграция с SimpleVoiceChat
- Игры в чате
